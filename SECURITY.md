# Güvenlik

Güvenlik açığını herkese açık issue olarak paylaşmayın. Depo sahibiyle GitHub Security Advisories üzerinden iletişim kurun.

- API anahtarları yalnızca `.env` içinde tutulur.
- Çekilen uzak içerik talimat değil, güvenilmeyen veri kabul edilir.
- Tam metin erişimi yayıncı lisansı ve erişim durumuna göre sınırlandırılır.
